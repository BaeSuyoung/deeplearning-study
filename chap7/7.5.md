## 7.5 seq2seq를 이용하는 애플리케이션  
1. 기계 번역 : 한 언어의 문장을 다른 언어의 문장으로 변환
2. 자동 요약 : 긴 문장을 짧게 요약된 문장으로 변환  
3. 질의 응답 : 질문을 응답으로 변환  
4. 메일 자동 응답 : 받은 메일의 문장을 답변 글로 변환
